NCAR/NSF C-130 Navigation, State Parameter, and Microphysics HRT (25 sps) Data
https://data.eol.ucar.edu/dataset/89.002

Summary
This dataset contains airborne measurements obtained from the NSF/NCAR
Research Aviation Facility (RAF) C-130Q Hercules aircraft (Tail Number N130AR)
during the VAMOS Ocean Cloud Atmosphere Land Study (VOCALS) which runs 15 Oct
- 15 Nov 2008. This data set contains the High-Rate (HRT, 25 sps) production
data merged with CVI and FO3 data.
