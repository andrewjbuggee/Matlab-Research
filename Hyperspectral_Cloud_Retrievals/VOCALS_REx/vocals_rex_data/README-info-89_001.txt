Navigation, State Parameter, and Microphysics LRT (1-sps) Data
https://data.eol.ucar.edu/dataset/89.001

Summary
This dataset contains airborne measurements obtained from the NSF/NCAR
Research Aviation Facility (RAF) C-130Q Hercules aircraft (Tail Number N130AR)
during the VAMOS Ocean Cloud Atmosphere Land Study (VOCALS). This data set
contains the Low-Rate (LRT, 1 sps) Flight Level production data merged with
the CVI, DMS, O3, PMS-2D, SO2, and TSURF data. Final CO data were added Oct,
2011.
